# Flappy Bird Clone

Flappy Bird clone made with Lua and Löve2D.

## Installation

You need git and [Löve2D](https://love2d.org/) installed.

```sh
$ git clone https://github.com/vanillaiice/flappybird-love2d
$ cd flappybird-love2d
$ love .
# or
$ make
```

## Game Controls

`Space` - Jump

`Enter` - Pause

`Backspace` - Restart

`Esc` - Quit
